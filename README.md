# aws-3tier-webapp
Deployment of a 3-Tier Web Application using AWS Services
# AWS 3-Tier Web Application

## Technologies Used

- AWS EC2
- VPC
- RDS
- Application Load Balancer
- Auto Scaling
- IAM
- Security Groups
- CloudWatch

---

## Architecture

Internet

↓

Application Load Balancer

↓

Web Tier EC2

↓

Application Tier EC2

↓

RDS MySQL

---

## Features

- High Availability
- Scalability
- Secure Network Design
- Database Integration
- Monitoring

---

## Implementation Steps

1. Created VPC
2. Configured Public and Private Subnets
3. Launched EC2 Instances
4. Configured Security Groups
5. Created RDS Database
6. Configured Load Balancer
7. Tested Application Connectivity
